# Microsoft Fabric Demos

This repository contains demo artifacts designed to illustrate various Microsoft Fabric capabilities and use cases. It serves as a collection of practical examples and implementations showcasing Fabric's features, starting with the foundational Fabric SQL demo.

## Project Overview

This repository demonstrates real-world scenarios and implementations using Microsoft Fabric, providing resources for learning, experimentation, and reference. Each demo includes code samples, scripts, and documentation to help you understand and implement Fabric solutions.

## Demos

### Fabric SQL Demo
The foundational demo showcasing Microsoft Fabric SQL capabilities, including data ingestion, transformation, and warehouse operations. This demo illustrates how to work with Fabric SQL for building data solutions.

## Repository Structure

- **fabric-sql-demo/** - The first demo featuring Fabric SQL implementation
  - **bronze/** - Raw data layer with data ingestion scripts
  - **silver/** - Transformed data layer with notebooks for data processing
  - **gold/** - Analytics layer with dimensional and fact tables
