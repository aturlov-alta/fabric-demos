# Silver: notebooks & ML view examples (Fabric Lakehouse)

This folder contains Jupyter notebooks that demonstrate creating and refreshing machine-learning-friendly materialized views (MLVs) from the prepared datasets.

**Files**

- `create_mlvs.ipynb` — Example notebook that creates MLVs from the dimensional model.
- `refresh_mlvs.ipynb` — Notebook that demonstrates refreshing/updating MLVs.

**Usage (in Fabric)**

- Open these notebooks in VS Code with Jupyter extension. The notebooks are not directly runnable. Copy and paste cell contents manually into Fabric Notebooks.
- See [FABRIC_SOLUTION.md](../FABRIC_SOLUTION.md) for the end-to-end context.
