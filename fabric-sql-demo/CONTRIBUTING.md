# Contributing

Thanks for your interest in improving this demo repository. A few guidelines:

- Please open an issue before making large structural changes.
- For code or SQL changes, include a short description and, if possible, a small test or sample inputs.

**Support and questions**

- Please use GitHub Issues for bug reports, questions, and feature requests.
- Discussions are intentionally not enabled; Issues are the single support channel.

**Maintainer**

- Primary maintainer: @aturlov-alta (GitHub). For all inquiries, please open an Issue.
